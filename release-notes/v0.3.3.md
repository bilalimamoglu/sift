## 0.3.3

### Improved
- Expanded deterministic benchmark coverage across `typecheck-summary`, `lint-failures`, `build-failure`, `audit-critical`, and `infra-risk` so `sift`'s agent-facing token reduction can be measured against a shared executable corpus.
- Improved `infra-risk` output for Terraform destroy plans with structured `destroy_count`, `destroy_targets`, and `blockers`, plus better evidence selection for real-world destructive plans and `prevent_destroy` failures.
- Improved the benchmark corpus with larger noisy wall cases for grouped presets so before/after reductions better reflect realistic CI and infrastructure output.

### Added
- Added a generic preset reduction benchmark runner and coverage that validate deterministic heuristics against a shared catalog of corpus cases.
- Added full-length example pairs under `examples/` so long raw logs and their reduced outputs can be inspected side by side for docs and demos.
- Added deterministic clean-pass handling for `audit-critical` when the input explicitly reports zero vulnerabilities.

### Changed
- Updated docs examples and provenance notes to reflect the expanded benchmark gallery and structured infra-risk output.
- Bumped package version to `0.3.3`.
