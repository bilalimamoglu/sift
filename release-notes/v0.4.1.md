## 0.4.1

### Improved
- Improved `sift exec` with a tiny TTY-only waiting notice for long-running child commands so interactive terminals no longer look frozen while the child process is still running.
- Improved release/docs polish so the public README, CLI guidance, and managed-agent guidance stay aligned with the current `rerun --remaining` behavior and the new child-command waiting notice.

### Added
- Added a small shared pending-notice helper for long waits, which now covers both provider follow-up and child-command execution paths without changing stdout, exit codes, or final reduced output.

### Changed
- Bumped package version to `0.4.1`.
